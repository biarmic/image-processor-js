# ImageProcessorJS

ImageProcessorJS is an image processing simulator where all image processing algorithms are implemented using pure JavaScript.

## Features
* Save image (Ctrl + S or &#8984; + S)
* Bit depth
* Dithering:
    * Floyd-Steinberg
* Filters:
    * Flip
    * Grayscale
    * Monochrome
    * Dichrome
    * Glass
    * Blur
    * Posterize
    * Solarize
